SISTEMA DE HISTÓRICO DE PRECIOS
================================

✅ INSTALADO Y FUNCIONANDO

CÓMO USAR:
----------

1. Cambias precios en tus archivos .js (como siempre)
   Ejemplo: src/data/categories/memorias.js
   price: 300000 → price: 320000

2. Ejecutas:
   npm run save-prices

3. El script detecta automáticamente los cambios y actualiza el histórico

4. Haces commit:
   git add .
   git commit -m "Actualizar precios"
   git push

DÓNDE SE MUESTRA:
-----------------
El gráfico aparece automáticamente en la página de detalle de producto,
entre la info del producto y los detalles técnicos.

Solo se muestra si hay 2 o más registros de precio.

ARCHIVOS IMPORTANTES:
---------------------
- scripts/save-price-history.js → Script que guarda el histórico
- src/hooks/usePriceHistory.js → Hook para leer datos
- src/components/PriceChart.jsx → Componente del gráfico
- public/data/price-history.json → Histórico guardado (163 productos)
